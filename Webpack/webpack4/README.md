# webpack4 + vue2 构建学习

## 安装

```js
// 注意各个插件版本
// webpack
yarn add webpack@^4.46.0 webpack-cli@^3.3.12 --dev

// css
yarn add style-loader css-loader --dev

// 图像
yarn add file-loader --dev

// 根据模板自动生成.html 清理dist文件
yarn add html-webpack-plugin@^3.2.0 clean-webpack-plugin@^3.0.0 --dev

// 热更服务
yarn add webpack-dev-server --dev

// webpack-merge
yarn add webpack-merge@^4.2.2 --dev
```
